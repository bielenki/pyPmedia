# pyPmedia
QGIS plugin to rainfall filling gaps and calculate a mean precipitation series by variable Thiessen Polygons or IDW

# Dependencies
The required libraries are:

GDAL

Geopandas

Fiona

RasterIO

pyproj

xlrd
